Available in the premium version of this module.
  • Groups recieved from SAML Response can be mapped to Jenkins Groups.
  • Select Add and then Group Mapping.
  • Add multiple group mapping in similar way.
  • Delete the added mapping using Delete button.

E.g.
Group Name in Jenkins Group Name in SAML Response
jenkins-admin(The groups in Jenkins) admin(The group received in SAML response)
jenkins-user(The groups in Jenkins) user(The group received in SAML response)
